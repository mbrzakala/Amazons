# Deliverables
Elements required to be allowed to pitch
Artifacts from each day.
Team members and their responsibilities (ideally in a ng-diagram form, but it’s not a hard requirement).
## Day 1 [20p]
Business analysis of the process - BPMN or Event Storming. Presented in the form of a Miro board, other similar tool or at least readable screenshot. Provide a link in the submission form.

Analiza biznesowa procesu - BPMN albo Event Storming. Zaprezentowane za pomocą Miro, innego podobnego narzędzia lub co najmniej czytelnego zrzutu ekranu. Do końcowego formularza należy wrzucić link.
## Day 2 [20p]
Use of the Design System with correct application of Design Tokens (colors, typography, spacing, radii, shadows, etc.) and maintaining interface consistency.

Wykorzystanie Design Systemu z poprawnym zastosowaniem Design Tokens (kolory, typografia, odstępy, promienie, cienie itp.) oraz zachowanie spójności interfejsu.
## Day 3 [20p]
Link to the repository with working code and instructions on how to run it.

Link do repozytorium z działającym kodem oraz instrukcjami jak go uruchomić.
## Day 4 [20p]
Link to the evaluation report coveringk the path from list of input scenarios to the idea evaluation and metrics.

Link do raportu z ewaluacji działania systemu pokrywającej ścieżkę od scenariuszy wejściowych do ewaluacji pomysłów i metryk.
## Day 5 [20p]
Link to the deployed, working application.

Link do dostępnej publicznie, działającej aplikacji.
## Pitch [25p]
You don’t have to create slides, but it’s highly recommended.
You don’t have to speak English, but it’s slightly suggested (at least for the slides as there will be one English speaking judge).
You have 5 minutes, you will see the clock the whole time and you will be interrupted mid-sentence when the timer goes off.
No Q&A session.

Nie musicie tworzyć slajdów, ale jest to wysoce zalecane.
Nie musicie mówić po angielsku, ale jest to lekko sugerowane (przynajmniej na slajdach, ponieważ w jury będzie jedna osoba anglojęzyczna).
Macie 5 minut, przez cały czas będziecie widzieć zegar i zostaniecie przerwani w połowie zdania, gdy czas się skończy.
Brak sesji pytań i odpowiedzi.
Submission form fields
Link to Day 1 artifact.
Link to Figma design.
Link to the code repository.
Link to the Day 4 artifact.
Link to the working app.
(optional) Link to the pitch presentation.

## Rating all the teams [25p]
On top of that we ask you to rate all the teams' presentations. You will get your points only if at least one member of your team will provide a rating for ALL the teams.
